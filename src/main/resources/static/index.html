Welcome to: Ryanir Interconnections Flights Api Rest

Hi, this flight service is exposed on following URL: '/ryainAirFlights/interconnections'

This service is expecting 4 mandatory parameters described below:

  • departure is a departure airport IATA code
  • departureDateTime is a departure datetime in the departure airport timezone in ISO format(yyyy-MM-ddTHH:mm)
  • arrival is an arrival airport IATA code
  • arrivalDateTime is a arrival datetime in the departure airport timezone in ISO format(yyyy-MM-ddTHH:mm)

For example like this: /ryainAirFlights/interconnections?departure=DUB&departureDateTime=2018-05-01T10:00&arrival=WRO&arrivalDateTime=2018-05-30T21:25

This will show a JSON format result, with flights found from departure airport at departure time or later, to arrival airport at arrival time or before

including flights with one stop of at least two hours in the same day.
